# **App Name**: LG Pay

## Core Features:

- Login Authentication: Secure user login using phone number and password.
- Registration Module: New user registration with phone number verification (OTP).
- Password Recovery: Allow users to recover their password if forgotten via OTP verification.
- User Agreement Confirmation: Require users to agree to the user agreement during registration.
- Help Center Access: Provide access to a help center for user assistance.
- Multilingual Support: Offer the portal in multiple languages.

## Style Guidelines:

- Primary color: Vibrant purple (#9400D3) to reflect innovation and financial technology. Chosen because the linear gradient on the body includes shades of violet, the dominant hue in the original code.
- Background color: Very light purple (#F0E6FF), almost white. Provides a clean backdrop that aligns with a light color scheme.
- Accent color: Pink (#FF69B4) to complement the primary color and provide visual interest for calls to action and important elements.
- Body and headline font: 'Poppins', a geometric sans-serif typeface.
- Use clean, modern icons for input fields and interactive elements.
- Maintain a clean, centered layout optimized for mobile devices, aligning with a contemporary design aesthetic.
- Incorporate subtle transitions and animations on button interactions and page transitions to enhance user engagement.
